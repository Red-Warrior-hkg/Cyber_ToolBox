# Process Killer



A Python utility that lists active user accounts with running processes and allows terminating all processes belonging to a selected user.



## Features

\- Enumerates running processes

\- Filters system accounts

\- Interactive user selection

\- Confirmation before termination

\- Handles inaccessible or terminated processes gracefully



\## Requirements

\- Python 3

\- psutil



\## Usage



```bash

pip install -r requirements.txt

python process\\\\\\\_killer.py

``` 



\## Notes



This project was built for learning purposes and to practice process management using Python and `psutil`. It is intended for use on systems you own or are authorized to administer.
